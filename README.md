# Taller de Nivelación PI a PII

Este repositorio contiene la solución del taller de nivelación PI a PII, el cual contiene parte teorica y parte practica en Java, organizadas con las siguientes instrucciones:

---

# Contenido del repositorio

- git.md → Respuestas sobre Git 
- programacion.md : Respuestas sobre Programación en Java
- Calculadora.java : Calculadora básica
- ContadorVocales.java : Conteo de vocales y consonantes
- InvertirCadena.java : Inversión de texto


---

# Instrucciones para ejecutar
1. Clonar el repositorio:
git clone URL_DEL_REPOSITORIO

2. Entrar a la carpeta:
cd Taller-nivelaci-n-PII

3. Compilacion:
javac NombreArchivo.java

4. Ejecutar:
Java NombreArchivo

---

# Conclusion

El desarrollo de este taller permitió reforzar conocimientos fundamentales sobre Git, Markdown y programación en Java, los cuales fueron abordados durante las clases. Estos aprendizajes constituyen una base importante para el curso de Programación 2 y para futuros proyectos académicos.
